Add Seals or Other Symbols from the Library


You can add symbols from one of SmartDraw’s many libraries. In SmartPanel, click the Library Selector menu button above the symbol gallery and choose the "Seals" library to open it in the gallery.

To add a symbol, click on it in the symbol gallery, then click in the Work Area where you'd like to add it.




Find More Symbols

If you can't find the symbol you want to add, you can open other libraries using the menu just above the gallery. This menu, called the Library Selector, may have a few other choices of library, which you can click on to open in the gallery.

If none of the library choices have the symbol you want to add, click More... in the Library Selector to search SmartDraw's libraries (How?).

Add Multiples of the Same Symbol

To add multiples of the same symbol, hold the Shift key while clicking the symbol in the symbol gallery.

You can then click repeatedly on the Work Area without having to click the symbol in the symbol gallery again. When you're done adding that symbol, press the Esc (or Escape) key.
