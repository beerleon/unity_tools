Add Text

You can Add Text:

  • In Table Cells
  • As Free-Floating Text

Add Text to Table Cells

To add text to a form block, or any other table, simply click in the cell to which you would like to add text and start typing. For more on how to work in table cells, see Work in Table Cells.

Add Free-Floating Text

To add free-floating text to the Work Area, click Add Text in SmartPanel and click wherever you want to add the text. When you've finished typing, click somewhere else or press the Esc (or Escape) key to leave text entry mode.

When you add free-floating text, a text box is created. You can treat this text box like any other shape. You can rotate it, change its color and size, and so on. If you delete all of the text from a text box and then click somewhere else, you delete the text box so that your Work Area is not cluttered with invisible shapes.
