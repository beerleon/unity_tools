Insert a Table

You can add a table to your visual using the Insert Table button in SmartPanel.

Click to open a menu where you can choose a preset size of simple table, or click Define to create a more custom table.

Clicking Define in this menu opens the "Add New Table" dialog box from where you can choose from a list of premade formats and set the height and width of the table. Click OK to insert the table.
