Add an Image

You can add an image:

  • Anywhere in the Work Area
  • Inside a shape or table

Add an Image Anywhere in the Work Area

Before adding an image to the Work Area, make sure no items are selected.

To add the image, click Add Image in SmartPanel and use the dialog box that appears to browse your files for the image you want to insert.

Select the image and click Open to insert it. Click on the image and drag it to position it as you like.

Add an Image Inside a Shape or Table

To add an image inside a shape or table, click on the shape or table cell to select it.

Then, click Add Image in SmartPanel, search for the image in your files, and click Open.

Replace an Image

You can replace an image by selecting it and inserting a new image.

Edit the Image

To edit the inserted image, use the Picture tab (How?).
