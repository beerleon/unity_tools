Select a Project Planning Visual Template

The first thing you'll need to do is open a visual template. These templates are specially designed for a particular type of visual, making their construction quick and easy!

In the Template Preview Area, to the left of this Help panel, you can click on a visual to start building, or click on a file folder icon to view even more options and examples.

Build a Visual

Once you have clicked on a visual template, you'll be able to start building using SmartPanel, a visual-specific set of controls designed to automate and facilitate the visual construction process as much as possible.

You will also have access to standard controls for coloring, styling, editing text, inserting various items, and much more.
