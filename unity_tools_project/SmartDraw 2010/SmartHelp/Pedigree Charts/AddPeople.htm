Add People

SmartDraw's Automatic Shape Connectors make adding shapes as easy as clicking the mouse.

You can add:

  • Parents
  • A Descendant

Add Parents

To add parents to a box representing a person in the visual, first click on the box to select it.

Then, click Add Parents in SmartPanel.

Two boxes are instantly added one level above the selected box in the tree, properly spaced and connected.


For fast results, start typing immediately after adding a shape to add text to that shape.


Insert a Descendant

Similarly, to add a descendant of a box representing a person, click on the box, then click Insert Descendant in SmartPanel.





A box is instantly added in the level below the selected box, properly spaced and connected.

To build your visual even faster, hold the Ctrl key and use the arrow keys to add a shape in that direction.
