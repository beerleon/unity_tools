Add People

SmartDraw's Automatic Shape Connectors make adding shapes as easy as clicking the mouse.

You can add:

  • An Ancestor
  • A Descendant
  • A Spouse
  • A Sibling

Add an Ancestor

To add an ancestor to a box representing a person in the visual, first click on the box to select it.

Then, click Add Ancestor in SmartPanel.

A box is instantly added above the selected box, properly spaced and connected.


For fast results, start typing immediately after adding a shape to add text to that shape.


Add a Descendant

Similarly, to add a descendant of a box representing a person, click on the box, then click Add Descendant in SmartPanel.

A box is instantly added below the selected box, properly spaced and connected.

Add a Spouse

To add a box representing a spouse of a person, first select that person's box.

Then, click Add Spouse in SmartPanel.

A box is added to the right of the selected box, and they are connected in brackets.


Add a Sibling

To add a sibling of a person, click on the box representing the person to select it. The box must already have a parent added in order to add a sibling.

Then click Add Sibling in SmartPanel.

A box is added to the right of the selected box, connected by the top edge to the boxes' parent(s).


To build your visual even faster, hold the Ctrl key and use the arrow keys to add a shape in that direction. The "Up" arrow adds an ancestor, the "Down" arrow adds a descendant, and the "Left" and "Right" arrows add a sibling in that respective direction.
