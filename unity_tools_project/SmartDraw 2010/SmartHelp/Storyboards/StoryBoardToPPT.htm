Build a PowerPoint® Presentation Using Your Storyboard

Once you create your Storyboard, you can use it to build a Microsoft PowerPoint® presentation.

Choose a PowerPoint® Theme

If you have some PowerPoint® themes, you can choose one for your presentation using the Theme menu button in the PowerPoint® group of SmartPanel.


Choose a Slide Layout

You can choose the layout of each slide.

Just click on a slide, or select several (How?), then choose a layout from the Layout menu in SmartPanel. Nothing will visibly change in SmartDraw, but when you build the presentation, the slides will reflect the chosen layout.

Build a Presentation

When you’re ready, click Build in the PowerPoint® group of SmartPanel.

A new presentation opens and each slide in the Storyboard becomes a slide in the presentation, with the elements appearing as bulleted text. If you insert any graphics, they also appear nicely laid out on the slide.
