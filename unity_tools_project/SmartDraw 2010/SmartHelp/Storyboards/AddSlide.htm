Add a Slide, Element or Sub-Element

SmartDraw's Automatic Shape Connectors draw lines for you and connect new symbols to others already on the Work Area, enabling you to build your visuals in record speed.

Add a Slide

Each slide you add represents a slide in the presentation you're outlining. If you use your storyboard to build a Microsoft PowerPoint® presentation (How?), SmartDraw will automatically add a slide in PowerPoint® for each slide on your storyboard.

To add a slide, click on the slide to the left of where you'd like to add a new slide and click Add Slide in SmartPanel. SmartDraw automatically adjusts the visual so the shapes are properly spaced.

To add slides, elements, and sub-elements even faster, hold the Ctrl key down and press an arrow key in the direction in which you would like to add the shape.


Add an Element

Each element you add to a slide represents a bullet point on that slide.

To add an element to a slide, click on the slide and click Add Element in SmartPanel. SmartDraw will automatically adjust the visual to accomodate the new shape.

Add a Sub-Element

A sub-element represents an indented bullet point of the element it is a subordinate of.

To add a sub-element to an element, click the element and then click Add Sub-Element in SmartPanel.

AutoStyle

You may notice that slides have the same color and style, while elements and sub-elements have different colors, shapes and styles. Coloring your visual this way helps the viewer to read it and SmartDraw does the work for you.

If you'd prefer not to use this coloring, click the AutoStyle checkbox button in SmartPanel before you start adding slides and elements.

You can change the colors using themes, quick styles, and more (How?).
