Create a Chart

Use SmartPanel to create a new chart in a variety of ways.

Create a New Blank Chart


As you can see, the template starts with a chart in the Work Area. If it's not the type you want to make, click Change Chart Type in SmartPanel and choose a new chart type from the menu.

Note: Using the Create New Chart button will insert a new chart in addition to the chart already in the Work Area.

Use Data from a Spreadsheet to Create a Chart


For faster data entry, you can copy and paste a spreadsheet directly to the Work Area. Copy the spreadsheet from the spreadsheet application (such as Microsoft Excel®) and click Paste Data from Clipboard in SmartPanel to paste the copied data into SmartDraw. A chart is generated from the pasted data.

Or you can import a spreadsheet. Click Import from Excel® File, browse to your file, and click Open. The File Import dialog opens. Make any necessary adjustments and click Import.

These options are also available on the Chart tab.

When choosing a chart type, remember that Pie charts and Relative Value charts only show one category at a time. If you switch to one of these two chart types from one of the other ones, you can choose which category to display by clicking Pie Data in the Chart Data group of the Chart tab.
