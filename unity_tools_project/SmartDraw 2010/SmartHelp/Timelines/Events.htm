Add, Edit or Remove an Event


Add an Event

To add an event to your timeline, simply click Add Event in SmartPanel.


Edit an Event

Once you have added an event, click on it to select it, or select several (How?).

Then, change the date, connector style, event shape, and whether the event attaches at the top or bottom of the timeline using those respective controls in the Events group of SmartPanel.

While the event is selected, you can add text to it simply by typing.

To build your timeline even faster, just double-click on the timeline to add a new event at that time. You can also drag an existing event to a new time.


Remove an Event

You can remove an event by selecting it and clicking Remove Event in SmartPanel. Or select it and press the Delete key.
