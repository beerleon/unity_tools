Add Employees, Assistants, and Co-Managers

SmartDraw's Automatic Shape Connectors draw the lines for you, making adding shapes as easy as clicking the mouse.

You can add:

  • An Employee
  • An Assistant
  • A Co-Manager

Add an Employee

To add an employee next to another employee, click the existing employee's box to select it.

In the Build group of SmartPanel, click one of the six buttons to choose a box format for the box you'd like to add. Then click Add Left, Add Right, Add Above or Add Below to add the shape in that respective direction.

The new box automatically connects to the box you selected, and all of the boxes in the visual adjust so they are all evenly spaced.


For even faster results, hold down the Ctrl key and press an arrow key to add an employee in that direction from the selected employee.


Add an Assistant

You can add a specialized box to represent an assistant. This box will branch off beneath and to the right of the person the assistant works for.

To add an assistant to an employee, select the employee and click Add Assistant in SmartPanel.

Add a Co-Manager

You can add a specialized box to represent co-managers. Since co-managers are equal in hierarchy, they are shown linked in a frame of branches.

To add a co-manager, click on the person who will be co-managing and click Add Co-Manager in SmartPanel.
