Import or Export an Org Chart

Import a File

You can create your visual quickly and easily by importing a .CSV or .TXT file of data - SmartDraw will read the file and create the visual for you! Click Advanced Options in SmartPanel and click Import Chart From Data. Choose a file in the dialog box that appears and SmartDraw will do the rest.


Export an Org Chart

You can also export your finished visual as a .TXT outline - click Export Chart as Data, also within Advanced Options and use the dialog box that appears to name and choose a location for your file. SmartDraw will format the file based on the visual you build.
