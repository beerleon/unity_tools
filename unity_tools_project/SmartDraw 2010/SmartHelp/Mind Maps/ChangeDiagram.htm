Convert Your Mind Map to a Project Chart or Timeline

You can quickly switch between Project Charts, Mind Maps, and Timelines so you don’t have to rebuild your data after creating one of these visuals.

To do this, click the Change Diagram menu button in SmartPanel and choose another visual.

Layer tabs will appear below the Work Area so you can quickly switch between visuals.

Any subtopic in a Mind Map becomes a subtask in a Project Chart, and vice versa. The Timeline will simply show main tasks, topics and milestones, but not subtasks or dependencies.

This is a great way to plan out a Project Chart before actually constructing it (How?).
