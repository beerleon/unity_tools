Select Items on the Work Area

You can select an item by clicking on it; once an item is selected, black or white square handles appear along its border or line.

Use the Shift Key

To select multiple items at once, hold the Shift key while clicking on each item in turn.

Trace a Box Using the Mouse

To select a group of items on the Work Area, click the Work Area and drag the mouse to trace a box around them.


Use the Select Button

You can use the Select button and menu in the Tools group of the Home tab for more selection options. Click the arrow beneath "Select" to view the menu. Select Multiple changes your cursor so that every item you click is selected and stays selected - just like when you hold the Shift key while clicking items.

Selecting All selects every item on the Work Area. Selecting All Shapes selects only symbols and anything else that is not a line. And selecting All Lines selects all of the lines on the Work Area.


Move Quickly from Item to Item

You can also move from item to item using the Tab key. To reverse directions, press the Shift key while pressing Tab.

To change the order in which you select items while pressing Tab in sequence, use the Bring to Front and Send to Back buttons in the Shape Layout group of the Design tab.

I Can't Select an Item on My Visual

If your visual uses layers, you might be unable to click an item if it is on another Layer. In this case, click the appropriate layer tab at the bottom of the Work Area so the item is on the top layer and you can select it.
