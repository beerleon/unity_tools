Insert a Web Page

You can quickly capture an image of any web page and insert it directly to the Work Area. This is better than taking a screen shot yourself because SmartDraw will capture the entire page, even if it scrolls below your screen.

To capture an image, click Web Page on the Insert tab, (or if you're creating a Web Page Annotation, click

in SmartPanel).

Type the URL into the bar near the top of the browser that appears. Once you've navigated to the page you'd like to capture, click Capture Webpage at the top of the window to add the screenshot to the Work Area.

SmartDraw automatically creates the layers "Web Page" and "Annotation" so you can write notes over the top of the web page without any interference.

For more on layers, see Layers.
