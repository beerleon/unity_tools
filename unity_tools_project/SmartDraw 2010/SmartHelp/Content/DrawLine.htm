Add a Line

SmartDraw automatically draws lines for you in most visual types, such as Flowcharts and similar visuals involving connected symbols. These lines are called Automatic Shape Connectors.

You can also manually draw a line with the Line tool.

To add a line, click the Line button in the Tools group of the Home tab. Click the menu button beneath the Line button to select a type of line:

A Straight Line enables you to draw a straight line anywhere on the visual. Click where you want to start the line, drag the length of the line, and then release the mouse button to complete the line.

The Curved Line is created in the same way as a straight line, but it creates an arc instead. You can adjust the depth of the arc by clicking on the finished line and dragging the black square handle in the middle of the line.

The Shape Connector line is useful when you want to connect shapes without using the automatic commands some visual types have. It creates three joined line segments at right angles to each other which you can stretch between two shapes.

The Curved Connector line serves the same purpose as the shape connector, except that it uses a curvier line rather than precise right angles.

The Freehand Line enables you to draw whatever you want as if you were drawing with a pen. SmartDraw smooths the curves for you, and you can make adjustments after you’re done drawing it by clicking and dragging handles along the line. You can also create a solid shape by joining the ends of the line together.

The Polygon Line allows you to create a line with as many angled bends as you want. Just click at every point you wish to have an angle at. You can create a solid shape by joining the ends of the line together.

There may also be a line tool in SmartPanel, depending on the visual you're working with. It may be titled slightly differently (e.g., "Lines", "Connect People As Parents", etc.).

For more on how to connect symbols with lines, see Connect Shapes with Lines.
