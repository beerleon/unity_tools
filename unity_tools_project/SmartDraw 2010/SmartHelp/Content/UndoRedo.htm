Undo or Redo an Action

You can Undo almost any action you make to a visual using the Undo button in the Quick Access Toolbar, found at the top left corner of the window. Clicking this button - or pressing Ctrl + Z - will undo the last action you perform.

If you undo an action, you can Redo it by clicking the Redo button, found immediately to the right of Undo, or press Ctrl + Y.

You can undo and redo up to 100 actions at a given time. If either of these buttons are gray, there is no previous action to undo or redo.
