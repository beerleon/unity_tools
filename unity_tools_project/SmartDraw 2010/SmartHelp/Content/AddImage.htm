Add an Image

You can add images anywhere on the Work Area. To add an image to the Work Area, click one of the buttons in the Get Images group of the Picture tab: From Camera, Picture Folder, Open Picture, or Capture Web.

To add an image to a shape or table, click on the shape or highlight the desired cell in the table to select it, then insert the image as above.

Edit the Image

You can use the Picture tab to edit any images you insert (How?).
