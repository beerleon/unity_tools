Add Free-Floating Text

When adding free-floating text to the Work Area, it's best to add the text in an area where there aren't any other symbols, and outside any frame of walls. Then, you can move the text (How?) to its position. This ensures that you actually add free-floating text rather than to a symbol or a room, which is more difficult to manipulate to fit your floor-planning needs.

To add text, click Text in the Tools group of the Home tab and click wherever you want to add the text. When you've finished typing, click somewhere else or press the Esc (or Escape) key to leave text entry mode.

When you add free-floating text, a text box is created. You can treat this text box like any other shape. You can rotate it, change its color and size, and so on. If you delete all of the text from a text box and then click somewhere else, you delete the text box so that your Work Area is not cluttered with invisible shapes.
