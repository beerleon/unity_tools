Use the Split Path Buttons

Add a Split Path

You can add two or more symbols branching from a single symbol, by using the Split Path buttons in SmartPanel.

Begin by clicking the symbol in the Work Area to which you'd like to add a split path. Next, choose a symbol in gallery in SmartPanel and click one of the directional Split Path buttons to add two of the selected symbol.






To add more than two splitting symbols, first add a split path as described above. Then choose a symbol in the gallery and press the Enter key once for each additional symbol you want to add.


Create a Split Path With Different Symbols

If you want to split a path with two different symbols, begin by adding a split path as above. Then, replace one of the symbols with a different symbol:

Join Paths

To change a split path to a straight line, select one of the involved shapes and click Join Paths. This will move the split shapes into a chain below the parent shape.
