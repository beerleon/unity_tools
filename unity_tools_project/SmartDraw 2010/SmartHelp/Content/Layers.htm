Layers

You can define more than one layer on your visual to separate the visual into different "pieces."

Think of a set of layers as a stack of transparent sheets of paper with symbols and lines on them. Layers are typically used to make complex visuals like floor plans where, for example, the walls may be on a different layer from the furniture or wiring. This is because layers keep items from interfering with each other.

Every visual template begins with at least one layer, called the Default layer. Each additional layer has a unique name and any layer can be moved within the "stack" to lie above or below any other layer. You can specify whether a layer is visible (i.e., whether the layer can be seen when it is not on top), and whether it is clickable (i.e., whether the symbols and lines in the layer can be manipulated even when the layer is not on top).

To create and manage a layer, use the Layers menu button in the Page Setup group of the Page tab. Click Define Layers to open a dialog box where you can manage a list of your layers, enabling you to quickly create, name, and edit all layers used in the visual. You can also use this menu to organize items within a layer.

When you have more than one layer, silver layer tabs with the layers’ names appear at the bottom of the Work Area.

Use these tabs to quickly switch between layers and set whether each layer is visible or clickable.
