Insert a Map

You can capture a custom map powered by Google® and insert it to your SmartDraw visual. To do this, click the Insert tab and click Map. This opens a window where you can select and format the map.


Set the Location and View

To set a map location and view, enter a location into the bar at the top of the window.

Use the Format controls to change the appearance of the map. Choose between a road map, satellite image map, or satellite image with roads superimposed in the Map Type menu. You can slide the slider freely to change the Zoom level of your map. Choose the size and shape of your map using the Map Size menu.


Add Overlays

You can add different types of overlays using the Overlays group. Click Add Data & Markers to import data from a file or paste data such as regional sales. Follow the instructions on the dialog box to add this data. SmartDraw will use the information it gathers from your data to place markers on the map.

There are checkbox buttons below the Add Data & Markers button. If you check a box, outlines of each country, state, county or zip code that is checked will appear on your map.

You can select a shape quality to determine how accurate the outlines are. Reduce the quality if you notice any significant delays.


An Outlined State

If you add markers or outlines, you can use the Background menu in the Format group to specify how much of the map is shown with respect to the outlines. You can also click the checkbox buttons to hide or show markers, overlays, and traffic.


Import the Map

Once you're done creating your map, click Import to SmartDraw to send the map to the Work Area.
