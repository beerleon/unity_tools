Export to Other Formats

SmartDraw automatically creates a .SDR file when you save your visual (How?). You can also save your visual by exporting it as one of many popular, sharable file types, such as PNG, PDF, JPEG, HTML, GIF, and several more.

To save your visual as one of these files, click the SmartDraw Button , located in the top left corner of the screen. Position your cursor over Export and click a file type in the list that appears.

A dialog will appear requesting information specific to that file type, such as where to save the file and what to name it. Make your selections and click Save (or Export for HTML).

A second dialog box may appear depending on the file type you choose. This box requests your preferences on the scale and resolution (in dots per inch) of the file and also tells you how much memory is required to export your visual.

Click OK to export your visual.
