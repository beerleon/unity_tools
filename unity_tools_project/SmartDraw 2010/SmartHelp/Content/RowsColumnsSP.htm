Add or Delete Rows and Columns

You can add or delete rows and columns to the table in the visual using the buttons in SmartPanel.

Add a Column

Begin by clicking a cell in the column to the left of where you'd like to add a new column.

Then click Add Column in SmartPanel to add a column to the right of the selection.

Add a Row

Begin by clicking a cell in the row above where you'd like to add a new row.

Then click Add Row in SmartPanel to add a row beneath the selection.

Delete a Column or Row

Begin by clicking the cell(s) of the row(s) or column(s) you'd like to delete.

Then click either Delete Column or Delete Row in SmartPanel.
