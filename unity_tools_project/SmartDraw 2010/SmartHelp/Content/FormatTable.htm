Format a Table

You can format any table you add in SmartDraw (How?) to change its structure or color. Begin by selecting the table and clicking Format Table in the Table Style group of the Table tab.

The "Format Table" dialog box opens, where you can preview and choose a format from a list, along with a few other options like the number of rows or columns for your table. Click OK to apply your selection.


Add Color to Cells

You can manually color the selected cells of any table by using the Fill button in the Table Style group of the Table tab.


Add Color and Style to Gridlines

To change the thickness, style, and color of gridlines, click on a gridline to select it. Then click Grid just below the Fill button and choose from the menu.

Add Effects

You can also add effects like gloss, shadow, or bevel to cells. Select the cells you want to change, then click Effects below the Fill button. There are many effects you can choose from.

Note: You cannot apply a perspective effect until after you are done with the visual because you won’t be able to use the controls anymore.
