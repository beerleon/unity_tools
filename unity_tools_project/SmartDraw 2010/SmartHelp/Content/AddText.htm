Add Text

You can Add Text:

  • On or Beneath Symbols
  • On Lines
  • As Free-Floating Text
  • To Table Cells

Add Text to a Symbol or Line

To add text to a symbol or along a line, double-click the symbol or line and begin typing.


If you're adding text to several symbols, you can move between symbols quickly and easily by pressing the Tab key. Press the Shift and Tab keys together to reverse directions.

You can change where and how text lies on a line once you've finished entering the text (How?).

Add Free-Floating Text

To add free-floating text to the Work Area, click Text in the Tools group of the Home tab and click wherever you want to add the text. When you've finished typing, click somewhere else or press the Esc (or Escape) key to leave text entry mode.

When you add free-floating text, a text box is created. You can treat this text box like any other shape. You can rotate it, change its color and size, and so on. If you delete all of the text from a text box and then click somewhere else, you can delete the text box so your Work Area is not cluttered with invisible shapes.

Many SmartPanels also contain an Add Text button, which functions in the same way as the Text button described above.


Add Text to Table Cells

To add text to a table cell, simply click inside the cell and start typing. For more on tables, see Working with Table Cells.
