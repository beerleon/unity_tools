Change the Connection Points of a Symbol

When you link a line to a symbol (How?), Connection Points appear along the symbol's border, indicating points you can attach the line to.

You can change the connection points of a symbol by selecting the symbol and clicking Connection Points in the Shape Properties group of the Design tab.

In the dialog box that appears, you can create Custom or Continuous connection points, (i.e., you can link to any point on the border of the symbol) or you can revert back to Default.
