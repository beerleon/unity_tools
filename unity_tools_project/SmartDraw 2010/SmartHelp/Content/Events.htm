Add, Edit, or Remove an Event

To add an event, use the Events group of SmartPanel. Here you can specify the date of the event, determine how the marker will attach to the timeline, and specify the shape of the event marker.

Once you've applied your settings, click Add Event and SmartDraw will automatically add the event. Double-click the text box of the event’s marker to label it.

To change the settings of an event you’ve already added, click on it to select it. Then click on the same options used to add the event in SmartPanel.

To remove an event, select it and click Remove Event in SmartPanel or select it and press the Delete key.
