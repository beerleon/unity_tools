Add Data to Your Chart

It's best to create a chart using the Chart visual template, and then to paste (How?) the chart into the main visual you're working on. This way you can use the Charts SmartPanel while you work.

If you insert a chart from a file or pasted data, you won't need to make too many data changes to your chart. If you add a new chart, you’ll need to add more data. There are several ways to do this.

Edit the Data Labels

You can show data labels in a variety of ways, by using the Data menu button in the Labels group of the Chart tab. If you choose to show data labels, you can click those labels and enter new values. The chart will automatically adjust to reflect the change.

Drag a Series by its Handles

When you select a series, black square handles appear on it. You can drag these handles to change the data value as well.

Edit the Data Table

If you’d rather work with a table, click the at the bottom right of your chart. Or click Show Data Table in the Chart Data group of the Chart tab to show the data table corresponding to your chart. Any changes made in the table are immediately reflected in the chart, and vice versa.
