Make a Website Map


Enter a URL

If you're mapping a website that's already online, making your Web Site Map is as easy as entering or pasting the URL into the Website bar in SmartPanel. Click Go and wait a few moments while SmartDraw does all the work!


Manually Build a Website Map

If you're building a map for a web site that has not been posted yet, you can use the Build group in SmartPanel, beginning with adding a Home Page. You can then add child pages to any selected shape by clicking the shape and clicking Add Child Page. SmartDraw automatically formats the shapes for you.


Delete a Shape

You can delete any shape you’d like by selecting it and clicking Delete Selected Page in SmartPanel. Be careful: any subordinate shapes (child pages) of the shape you delete are deleted as well.

Show or Hide Redundant Links

If a link appears in several places on your map, you can check or uncheck Show Redundant Links in the Options group of SmartPanel to hide or reveal the repetitions.


Get Screenshots or Page Titles

Each shape on your Web Site Map includes an image and a link. You can get the title and a screenshot of each page by selecting a shape or several shapes (How?) and clicking Get Page Title or Get Screenshots in SmartPanel. SmartDraw will go to each page and acquire the requested information, replacing the image and/or link in each selected box.

Note: It may take a few moments for SmartDraw to complete these tasks since it must rely on the Internet.
