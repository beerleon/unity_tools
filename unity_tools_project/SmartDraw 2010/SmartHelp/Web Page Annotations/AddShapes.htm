Add Symbols for Annotation


You can add symbols from the "Annotations" library to annotate the web page you have inserted (How?).

Just click on a symbol in this gallery, then click again in the Work Area where you would like to annotate something.



While the symbol is still selected, simply start typing to add text.

Layers

SmartDraw automatically creates the "Annotation" and "Web Page" layers when you insert a web page so that the annotation symbols do not interfere with the web page.

When you click on a symbol in the gallery, SmartDraw automatically puts the "Annotation" layer on top so that all your symbols are on the same layer.

Click on the "Web Page" layer tab to temporarily hide the annotation symbols you have added.
