Import a File or Directory

Import a Directory

SmartDraw will automatically search for a network directory on your computer, so a directory tree may be visible in the Import group of SmartPanel when you open the visual template.

If you wish to map another directory or if SmartDraw didn’t find anything, click either File or Directory to choose another path.

Decide which attributes of the directory you wish to map, and click the checkbox buttons for each of those entries.

Filter the Results

To help you find the attributes to map, you can filter the directory using the Filter Attributes group of SmartPanel. Click the checkbox buttons you want to see and check Filter Results. The directory is immediately filtered.


Build the LDAP Visual

Once you've checked all the attributes you want to add, click Build in SmartPanel. It may take a few seconds for SmartDraw to build the visual if it is a large directory.
