Add a Decision or Unknown

SmartDraw's Automatic Shape Connectors make adding shapes as easy as clicking a mouse.

Add a Decision


To add a decision to a shape on your tree, first select the shape. Then click Add a Decision in SmartPanel.

Two shapes are automatically added, branching from the shape you select, with "Yes" and "No" labels on the branches. Click on these labels and begin typing to edit them.


To add decisions even faster, just select a shape, then hold down the Ctrl key and press the arrow key in the appropriate direction (depending on the selected direction of the visual).

Add an Unknown

An unknown is part of a decision involving more than two options - you should begin by adding a decision, then adding unknowns to create three or more options.

To add an unknown to your tree, first select a shape which is connected to the same shape you'd like to add the unknown to. Then, click Add Unknown in SmartPanel.

A single shape is added next to the shape you select with a placeholder label on its branch. Click this label and begin typing to edit the text.


To add unknowns even faster, select a decision as above, and press the Enter key. An unknown is instantly added in the same level as the selected shape.
