Add More Symbols

In addition to adding doors and windows (How?), you can add symbols representing furniture, appliances, evidence, plants, and more. SmartDraw has hundreds of symbols for you to choose from.

Begin by choosing a library. In SmartPanel, click the menu button, called the Library Selector, above the symbol gallery.

In the menu, you can either choose another library title, or click More to open the More Symbols dialog box. In this dialog box, you can search for symbols using keywords:

Or browse the libraries using the explorer tree:

Click on a library in the list to preview it. When you have found the library you want, click on it and click Open Library, then OK. The library opens in the gallery in SmartPanel.

To add a symbol, just click on it in the gallery, then click again in the Work Area where you would like to add it.




You can rotate and flip symbols so that they are positioned correctly (How?).
