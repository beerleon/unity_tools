Set Up Your Project Chart

Before adding tasks to your Project Chart, you should set up the layout so that it works well with your data.

Set the Timescale


Your visual begins with a "Months with Days" timescale by default. You may begin creating your visual and realize that a different timescale may be more appropriate for your project. You can switch timescales at any time by clicking Timescale in SmartPanel.

When choosing a timescale, think about how long the project will be. If it will only be a few weeks, choose a timescale like "Week with Days" or "Months with Days". If your project will last years, choose a timescale like "Quarter with Months" or "Year with Months".

This choice will affect your ability to read the chart, as well as the width of the chart, which will matter when you want to print your visual (How?).

Set the Start and End Dates of the Project


To set the start and end dates, click Timeframe in SmartPanel. In the dialog box that appears, you can enter the start and end dates of the range of time to be shown in the visual, as well as the actual start date of the project.

Once you've selected the desired options, click OK to apply them to the visual.
