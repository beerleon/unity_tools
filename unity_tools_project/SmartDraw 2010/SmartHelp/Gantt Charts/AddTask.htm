Add Tasks, Subtasks, and Milestones

Add a Task

You can add a task to your Project Chart by either clicking Add Task in SmartPanel, or adding information to a row in the table section of the visual.

Once you create a task, you can change its dates and duration by editing the text in the table or clicking the task bar and dragging the handles on either side of it.

Add a Subtask

You can indicate that a task is a subtask of another task by indenting it. To do this, click on the task and click Indent Task in SmartPanel.

SmartDraw will renumber the indented task so it has a decimal value. For example, if you indent the task below Task 1, the subtask will be numbered 1.1. The task bar of the upper task also changes shape to indicate that it contains subtasks below it.

If a task is indented too far, you can outdent it by clicking Outdent Task in SmartPanel while that task is selected.

Your Project Chart may grow quite large. If it has subtasks, you can collapse the main task to temporarily hide its subtasks. This helps to condense your visual. To do this, click the next to the main task’s number. To show the subtasks again, click the next to the main task’s number.

Add a Milestone

You can also add milestones. To do this, click Add Milestone in SmartPanel. You can alter the date at which the milestone occurs, but it must have the same end date as the start date since it’s just a point.

If you’d like to add the milestone in the midst of the tasks you've already added, click on the task immediately above where you’d like to add the milestone before adding it.

To quickly change a task to a milestone, simply change the Duration to 0. You can also change a milestone to a task by increasing the Duration.


Remove a Task, Subtask, or Milestone

To remove a task, subtask or milestone, click on its cell in the chart and then click Remove Task in SmartPanel.
