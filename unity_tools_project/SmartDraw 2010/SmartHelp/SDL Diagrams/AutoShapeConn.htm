Add Symbols

Automatic Shape Connectors enable you to build your visuals in record speed. When working with Automatic Shape Connectors, each symbol you add automatically connects to another symbol already on the Work Area. This eliminates the need to draw lines between every symbol on your visual.

Begin by selecting a symbol in the Work Area.

Click a symbol in the symbol gallery in SmartPanel and use the "Add" buttons to add it in the desired direction from the selected symbol on the Work Area.

If a symbol is already connected to the selected symbol in the direction you want to add a new symbol, the new symbol will be added between the selected symbol and the connected symbol. The symbols shift to make room for the new symbol.


To add symbols even faster, choose a symbol in the gallery, then hold the Ctrl key and press the arrow keys to add the symbol in the desired direction (for example, Ctrl + right arrow key adds the symbol to the right).

Can’t find the symbol you want? Click on the Library Selector menu button above the symbol gallery to find more libraries (How?).

You can also add library symbols and shapes without connecting them (How?).
